Welcome to Space Simulator project

This is a siple simulator of gravity flying in the space. It operates with planets, satellites (planets of low mass) and stars (planets of big mass) The standard equation of movement is solved by Kutta-Runge method.

This software is based on Pythin3.3 and PyGame1.9

For starting the simulator please use command line:

python3.3 main.py -f .ini

or

python3.3 main.py -h

Some configuration files are in the root folder.

During the simulation you can use keyboard:

  • q or ESC = Exit
  • p or SPACE = Pause

Simulation stops when two or more bodies crashes or maximum distance between two bodies getting very big.

Have a fun :-)
